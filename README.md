# digital-literacy-project
Project Overview - 
This repository contains my Digital Literacy Portfolio created as part of the CSE0001 course at VIT Bhopal University. As a Student Digital Ambassador, this project demonstrates my understanding of digital tools, online safety, professional communication, and cyber awareness.
 Repository Structure - 

 Task 1 – Presentation
Created a digital literacy infographic using Canva
Covers digital literacy, tools, and safe internet practices

 Task 2 – Digital Portfolio
Platforms created:
GitHub
LinkedIn
Kaggle
Includes screenshots of profiles and setup

Task 3 – Platforms
Completed beginner challenge on Hacker Rank
Created a Google Form quiz on digital literacy
Added screenshots of challenge and form responses

Task 4 – Email Etiquette
Drafted two professional emails:
Request for assignment extension
Internship application
Created a social media Do’s and Don’ts checklist.

Task 5 – Cybercrime Awareness
Case study on UPI/Online Payment Fraud
Created a “Stay Safe Online” prevention checklist

Key Learnings - 
Importance of digital safety and awareness
How to build a professional online presence
Effective email communication skills
Understanding cyber threats and prevention.

Conclusion - 
This project helped me develop essential digital literacy skills required for academic and professional growth. It also improved my awareness of online safety and responsible digital behaviour.
